# Full Stack Blog App — React & Express

##  Overview

This project is a **full stack blog application** that demonstrates real-world frontend–backend integration.
Users can create, view, edit, and delete blog posts through a clean React interface powered by an Express REST API.

The project showcases core full-stack development skills including API design, state management, async data handling, and responsive UI architecture.

##  Live Features

 Create new blog posts
 View all posts dynamically
 Edit existing posts
 Delete posts
 REST API integration
 Loading state handling
 Clean responsive UI
 Real-time UI updates after CRUD actions
 Full frontend ↔ backend communication

## Application Flow

1. React app loads existing posts from API
2. User creates a new post via form
3. Post saved on Express server
4. UI updates instantly
5. User can edit or delete posts
6. Loading state improves UX during fetch
7. Data stays in sync between frontend and backend
